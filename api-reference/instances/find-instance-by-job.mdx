---
title: "Find Instance by Create Job"
api: "GET /v2/instance/find_by_create_job/{job_id}"
description: "Retrieve an instance by its create job ID"
---

When you create an instance using the [Create Instance](/api-reference/instances/create-instance) endpoint, a job is returned. This endpoint allows you to find the resulting instance once the job completes.

## Path Parameters

<ParamField path="job_id" type="string" required>
  The ID of the create job returned when the instance was provisioned.
</ParamField>

## Response

<ResponseField name="uuid" type="string">
  The unique identifier for the instance.
</ResponseField>

<ResponseField name="short_uuid" type="string">
  A shortened version of the instance UUID.
</ResponseField>

<ResponseField name="name" type="string">
  The display name of the instance.
</ResponseField>

<ResponseField name="hostname" type="string">
  The hostname of the instance.
</ResponseField>

<ResponseField name="server_id" type="string">
  The server ID for the instance.
</ResponseField>

<ResponseField name="project_id" type="string">
  The project ID the instance belongs to.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the instance.
</ResponseField>

<ResponseField name="management_ip" type="string">
  The management IP address of the instance.
</ResponseField>

<ResponseField name="provision_date" type="string">
  The date when the instance was provisioned.
</ResponseField>

<ResponseField name="create_job" type="string">
  The job ID for the create job of this instance.
</ResponseField>

<ResponseField name="core" type="object">
  The core/plan configuration of the instance.
</ResponseField>

<ResponseField name="facility" type="object">
  The facility where the instance is located.
</ResponseField>

<ResponseField name="image" type="object">
  The operating system image installed on the instance.
</ResponseField>
