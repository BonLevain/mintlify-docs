## Overview

Returns a list of all available datacenter facilities where you can deploy servers. Each facility includes its code, human-readable title, and description.

<RequestExample>

```bash cURL
curl https://api.dallas-idc.com/v2/facility \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```python Python
import requests

response = requests.get(
    "https://api.dallas-idc.com/v2/facility",
    headers={"Authorization": "Bearer YOUR_API_TOKEN"}
)
print(response.json())
```

```javascript JavaScript
const response = await fetch('https://api.dallas-idc.com/v2/facility', {
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN'
  }
});
const data = await response.json();
```

</RequestExample>

## Response Fields

<ResponseField name="facility_name" type="string">
  The unique facility code used when creating instances or checking stock (e.g., `DFW1`, `LAX1`)
</ResponseField>

<ResponseField name="facility_title" type="string">
  Human-readable title including city, state/country, and facility code
</ResponseField>

<ResponseField name="facility_description" type="string">
  Short description of the facility
</ResponseField>

## Available Facilities

### North America

| Code | Location | Description |
|------|----------|-------------|
| DFW1, DFW2 | Dallas, TX | Primary Texas datacenter |
| LAX1, LAX2 | Los Angeles, CA | West coast datacenter |
| SLC1, SLC2 | Salt Lake City, UT | Mountain region datacenter |
| JFK1, JFK2 | New York, NY | East coast datacenter |
| MIA1, MIA2 | Miami, FL | Southeast datacenter |
| ORD1, ORD2 | Chicago, IL | Midwest datacenter |
| IAD1 | Washington, DC | DC metro area datacenter |
| PDX1 | Portland, OR | Pacific Northwest datacenter |

## Use Cases

- **Regional Selection**: Choose the facility closest to your users for optimal latency
- **Capacity Planning**: Identify available facilities before provisioning servers
- **Multi-Region Deployment**: Deploy across multiple facilities for redundancy
- **Compliance**: Select facilities in specific geographic regions for data residency requirements
