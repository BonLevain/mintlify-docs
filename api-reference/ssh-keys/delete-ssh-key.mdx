---
title: "Delete SSH Key"
openapi: "DELETE /v2/sshkey/{name}"
---

Delete an SSH key from your account by its name.

## Path Parameters

<ParamField path="name" type="string" required>
  The name of the SSH key to delete.
</ParamField>

## Response

Returns a `202 Accepted` status on successful deletion.

<RequestExample>
```bash cURL
curl -X DELETE "https://api.dallas-idc.com/v2/sshkey/my-key" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```python Python
import requests

response = requests.delete(
    "https://api.dallas-idc.com/v2/sshkey/my-key",
    headers={"Authorization": "Bearer YOUR_API_TOKEN"}
)
print(response.status_code)
```

```javascript JavaScript
const response = await fetch("https://api.dallas-idc.com/v2/sshkey/my-key", {
  method: "DELETE",
  headers: {
    "Authorization": "Bearer YOUR_API_TOKEN"
  }
});
console.log(response.status);
```
</RequestExample>
