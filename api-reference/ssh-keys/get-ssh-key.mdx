---
title: "Get SSH Key"
openapi: "GET /v2/sshkey/{name}"
---

Retrieve details for a specific SSH key by its name.

## Path Parameters

<ParamField path="name" type="string" required>
  The name of the SSH key to retrieve.
</ParamField>

## Response

Returns the SSH key object with the following properties:

<ResponseField name="id" type="string">
  The unique identifier of the SSH key.
</ResponseField>

<ResponseField name="name" type="string">
  The name of the SSH key.
</ResponseField>

<ResponseField name="fingerprint" type="string">
  The fingerprint of the public key.
</ResponseField>

<ResponseField name="comment" type="string">
  The comment associated with the public key.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.dallas-idc.com/v2/sshkey/my-key" \
```
