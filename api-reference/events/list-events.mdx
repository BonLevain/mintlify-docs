---
title: "List Event Logs"
api: "GET /v2/event"
description: "Get and filter event log list"
---

Retrieve a list of event logs with optional filtering by object type, object ID, and time range.

## Query Parameters

<ParamField query="type_id" type="string">
  ID of the object associated with the event.
</ParamField>

<ParamField query="type" type="string">
  The object type of event (e.g., `Instance`, `Project`).
</ParamField>

<ParamField query="start" type="string">
  The start time to look back. Defaults to 1 month if not specified.
</ParamField>

## Response

<ResponseField name="events" type="array">
  An array of event log objects.
  <Expandable title="Event Log Object">
    <ResponseField name="uuid" type="string">
      The unique identifier for the event.
    </ResponseField>
    <ResponseField name="event" type="string">
      The type of the event.
    </ResponseField>
    <ResponseField name="created_at" type="string">
      The timestamp when the event occurred.
    </ResponseField>
    <ResponseField name="message" type="string">
      A human-readable message describing the event.
    </ResponseField>
    <ResponseField name="username" type="string">
      The user who triggered this event.
    </ResponseField>
    <ResponseField name="data" type="string">
      Additional event data in JSON format.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://api.dallas-idc.com/v2/event?type=Instance&type_id=abc123" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

```python Python
import requests

response = requests.get(
    "https://api.dallas-idc.com/v2/event",
    headers={"Authorization": "Bearer YOUR_API_TOKEN"},
    params={
        "type": "Instance",
        "type_id": "abc123"
    }
)

events = response.json()
```

```javascript JavaScript
const response = await fetch(
  "https://api.dallas-idc.com/v2/event?type=Instance&type_id=abc123",
  {
    headers: {
      Authorization: "Bearer YOUR_API_TOKEN",
    },
  }
);

const events = await response.json();
```
</RequestExample>
