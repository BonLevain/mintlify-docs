---
title: "Get Project Subnets"
openapi: "GET /v2/project/{project_id}/subnet"
---

Retrieve the IP subnets allocated to a specific project.

## Path Parameters

<ParamField path="project_id" type="string" required>
  The unique identifier (UUID) of the project to query.
</ParamField>

## Response

Returns an array of [IpBlock](/api-reference/schemas/ip-block) objects containing:

- Network block with CIDR notation
- Network address and netmask
- Device assignment information
- Network type (public or private)
- Assignment type (direct or floating)

## Example Request

```bash
curl -X GET "https://api.dallas-idc.com/v2/project/{project_id}/subnet" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

## Use Cases

- **Network Inventory**: List all IP subnets allocated to a project
- **Capacity Planning**: Review IP allocation across the project
- **Network Management**: Understand which subnets are available for assignment to instances
- **Compliance Reporting**: Document IP address usage within a project
