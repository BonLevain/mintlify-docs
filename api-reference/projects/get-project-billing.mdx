---
title: "Get Project Billing History"
api: "GET /v2/project/{project_id}/billing"
description: "Retrieve the billing history for a project"
---

## Path Parameters

<ParamField path="project_id" type="string" required>
  The unique identifier of the project
</ParamField>

## Response

Returns an array of invoice objects representing the project's billing history.

<ResponseField name="id" type="string">
  The unique identifier of the invoice
</ResponseField>

<ResponseField name="timestamp" type="string">
  The timestamp when the invoice was created
</ResponseField>

<ResponseField name="type" type="string">
  The type of invoice (Credit or Debit)
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the invoice
</ResponseField>

<ResponseField name="description" type="string">
  A description of the invoice
</ResponseField>

<ResponseField name="amount" type="string">
  The total amount of the invoice
</ResponseField>

<ResponseField name="forpackage" type="string">
  The package associated with the invoice
</ResponseField>

<ResponseField name="forinvoice" type="string">
  The invoice associated with this invoice (in the case of a Credit)
</ResponseField>
