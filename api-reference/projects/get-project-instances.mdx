---
title: "Get Project Instances"
api: "GET /v2/project/{project_id}/server"
description: "Get a project's instances"
---

Retrieve all instances (servers) associated with a specific project. This endpoint returns detailed information about each instance provisioned within the project.

<ParamField path="project_id" type="string" required>
  ID of the project to return instances for
</ParamField>

## Response

Returns an array of instance objects containing the following fields:

<ResponseField name="uuid" type="string">
  The unique identifier for the instance
</ResponseField>

<ResponseField name="short_uuid" type="string">
  A shortened version of the UUID for convenience
</ResponseField>

<ResponseField name="name" type="string">
  The display name for the instance
</ResponseField>

<ResponseField name="hostname" type="string">
  The hostname assigned to the instance
</ResponseField>

<ResponseField name="server_id" type="string">
  The server ID for the instance
</ResponseField>

<ResponseField name="project_id" type="string">
  The project ID the instance belongs to
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the instance
</ResponseField>

<ResponseField name="management_ip" type="string">
  The management IP address of the instance
</ResponseField>

<ResponseField name="provision_date" type="string">
  The date the instance was provisioned
</ResponseField>

<ResponseField name="core" type="object">
  The core (hardware configuration) details for this instance
</ResponseField>

<ResponseField name="facility" type="object">
  The facility (datacenter) where this instance is located
</ResponseField>

<ResponseField name="image" type="object">
  The operating system image installed on this instance
</ResponseField>
