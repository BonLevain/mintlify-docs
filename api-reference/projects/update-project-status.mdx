## Overview

Updates the status of a project. This endpoint allows you to change the operational status of a project, which can affect the availability and behavior of resources within the project.

## Path Parameters

<ParamField path="project_id" type="string" required>
  The unique identifier of the project to update (e.g., `PROJ-9bf66e73` or the full UUID)
</ParamField>

<RequestExample>

```bash cURL

  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/x-www-form-urlencoded"
```

## Use Cases

- **Project management**: Update the operational state of a project
- **Maintenance workflows**: Change project status as part of infrastructure maintenance procedures
- **Lifecycle management**: Manage project states through different phases of use

## Notes

- Returns a `200 OK` status code when the update is successful
- Invalid status values will return a `422 Unprocessable Entity` error
- Changing project status may affect the availability of resources within the project
